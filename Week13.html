For this blog I wanted to learn how to do requests other than GET so I could use DELETE and POST on my project3. After watching a video and reading a few articles I was able to figure out how to make a POST request with my project2. Below is some working code that is currently in my project3. 

            const requestOptions = {
                method: 'PATCH',
                headers: { 'Content-Type': 'application/json' },
                body: JSON.stringify({"question": question, "answer1": answer1, "answer2": answer2, "answer3": answer3, "answer4": answer4, "correct_ans": correctAns})
            };
       
            fetch(`https://carter-gilbert.me:8443/questions/${id}`, requestOptions)
                .then(response => response.json())

DELETE requests or any request type can be done in a very similar manner, just ensure the method is correct and the header and body jsons have the correct data. The fetch request is then very similar to what we have done previously with GET, however, now we also pass it the requestOptions we created above. Finally, we capture the return response as a json which we can do whatever we want with.
