This week in class we have been studying JavaScript and doing some API calls to retrieve JSON lists. For some of the activities we will then take data from the lists and put it onto the screen using document.body.append. Before getting into JavaScript, I looked and didn’t see anything exciting online, I decided to do a simple animation for my title. These have been my favorite so far so I figured after taking last week off from my blog I would start with one more animation. This animation was relatively simple. For the week it just moves the letters into place from the top and makes them less opaque while for October 8th essentially, it just goes letter by letter and makes the letter less opaque and rotates it from upside down. It makes a pretty playful effect that I think suited this blog post.

Here is a code snippet from the 2nd animation.

    .two span {
        color: orange;
        opacity: 0;
        transform: translate(200px, -100px) scale(2);
        animation: ballDrop .3s forwards;
    }
    
    @keyframes ballDrop {
        60% {
            transform: translate(0, 20px) rotate(-180deg) scale(.5);
        }
        100% {
            transform: translate(0) rotate(0deg) scale(1);
            opacity: 1;
        }
    }

We can see from this that we are using the ball drop animation and each letter takes .3 seconds to complete the animation. As I said above, it sets the letters to be small and upside down, then makes each letter less opaque as it rotates and moves them to the correct position.

Then I decided I want to make a JavaScript button that would replay the animation when you click it. So, I followed the tutorial and made the two buttons that you see above. The first one will trigger the first animation while the second will trigger the second. The button works by finding an element with class ‘repeat’, which I have assigned to both animations, then it finds the animate attribute for the desired div before finally readding it which triggers the animation again.
