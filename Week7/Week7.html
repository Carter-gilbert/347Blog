This week we have continued to study JavaScript and began to make and use endpoints in express. I wanted to loop back to something we had mentioned earlier when we were working on project 1, forms. We have made forms before, but I wanted to learn how to accept the data and save it as a json file. What we need to do is make a form, as we have in the past but assign an id so that we can grab the value in JavaScript. For example: 


            <form>
                <label for="name">Name:</label>
                <input type="text" id="name" name="name" value="Name">
                <label for="email">Email:</label>
                <input type="email" id="email" name="email" value="Email">
            </form>

Notice how each part of the form has a id= so that when we code the JavaScript we can grab each of these elements and extract their values. We can use the getElementByID in JavaScript to get the elements by their ID name or email. Lastly we append this data to an array so that it can be stored. 

            data.append("name", document.getElementById("name").value);
            data.append("email", document.getElementById("email").value);
