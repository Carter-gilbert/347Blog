Week 1 - September 3rd

 This week in class we focused on flexboxes and how to move and arrange contents in new ways. The lab this week really helped me learn a lot about them and I can see their usefulness. I wanted to learn a few more practical ways to apply this, so I started by learning how to do split box navigation. As you can see below the Home button and the other buttons are split using the display attribute set to flex and the Home button is pushed to the left by using a simple automatic margin.

 This is a simple-looking nav bar, because it is a proof of concept, but I can imagine how this would be useful in many ways on a variety of websites. On this website it makes the home page very easy to find rather than it being in a list with the other blog posts. I want to learn how to style and format elements such as this navigation bar so when I begin to focus on styling for my static website I can make it look the way I envisioned.

 I also wanted to find a new animation to create. I wanted something that would fit into my PA-01 project and would not stick out or feel like I forced it in. I decided to try to find some social media buttons, after some searching, I found that most used SCSS, so I kept looking until I found something simpler that used CSS.





 As you can see when you hover over the button it highlights in the social media platform’s color and has a cool shadow effect. The tutorial did not have clickable icons so I was able to modify the code so that when you click on an icon it takes you to Twitter, Google, or Instagram's website respectively. Creating these animated buttons makes the website look much more professional and being able to link one’s personal social media in a cool way, rather than having a static icon, leads to a higher likelihood of the viewer clicking on it and following them on social media.
